The {{ result.name }} runs on a Microsoft Windows 11 Professional computer, or Linux x86_64 computer. Supported operating systems and Linux distributions include:

MacOS operating system unsupported
Unfortunately, MacOS is not supported with OpenShift Local since MacOS became ARM64 architecture.
Recommended computer specifications

If you are purchasing a laptop or desktop computer, I would suggest the following specs:

  1. At least 64 GiB RAM. Laptops with at least 64 GiB RAM have enough to run OpenShift Local and Spine Programming microservices.
  2. At least 12 CPUs. Laptops with at least 12 CPUs have enough to run OpenShift Local and Spine Programming microservices.
  3. At least 500 GiB solid state drive (SSD) storage. 1 TiB is enough storage for most developer's applications and documents.
  4. No GPUs are required for Spine Programming.
