System requirements

Here are the bare minimum system requirements for deploying the {{ SITE_NAME }} platform on your own computer. If you have more resources than this, then your system will perform even better.

System architecture x86_64 — computers that run Microsoft Windows are all compatible
Memory requirements 32 Gi RAM — this is the minimum amount of recommended memory.
CPU requirements 8 logical processors — if you have more processors than this, then you will be able to do more multitasking on your computer at the same time.
